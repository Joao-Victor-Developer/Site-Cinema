🎬 Cine Finder

Cine Finder is a web application developed to simplify the search and discovery of movies, series, and cinematic information. It uses modern APIs, PHP for backend logic, MySQL for persistent storage (via phpMyAdmin), and an intuitive interface to deliver a smooth and engaging experience for exploring the audiovisual world.

📖 About the Project

Cine Finder was designed to provide users with:

Search for movies and series by title

View synopsis, cast, release date, ratings

Browse categories and current trends

Save favorite titles with authentication

Register/login through PHP + MySQL integration

✨ Features

🔍 Smart search with auto-suggestions

🎞️ Detailed information for each title

⭐ Integration with APIs (TMDb or OMDb)

📱 Responsive design (mobile-friendly)

💾 Save favorites in database (MySQL)

🔐 User authentication system (PHP + MySQL)

🧠 Technologies Used

Runs locally with USBWebServer, which includes Apache, PHP, and MySQL (with phpMyAdmin).

Frontend: HTML5, CSS3, JavaScript ES6+

Backend: PHP 7+

Database: MySQL (via phpMyAdmin)

Libraries/Tools:

Axios for API requests

TMDb API or OMDb API

Bootstrap or TailwindCSS

🛠️ Installation

Download USBWebServer

USBWebServer Download

Clone the repository

git clone https://github.com/your-username/cine-finder.git
cd cine-finder


Move the project folder into the root/ directory of USBWebServer.

Start USBWebServer and ensure Apache + MySQL are running.

🗄️ Database Setup

Open phpMyAdmin:

http://localhost/phpmyadmin


Create a new database (example: cinefinder_db).

Import the SQL file located in the /database folder of the project.

Update the database connection settings in conexao.php (or equivalent config file).

🚀 How to Use

Open the browser and go to:

http://localhost/cine-finder


Register or log in to access full functionality.

Use the search bar to find movies or series.

Browse results and save your favorites.
